
# dogcheck
The famous DOGCHECK from Toby Fox's games. Now in a web format with customisation. Normally it will randomly choose a message and display it on the webpage but you can change that. There is also a 5% chance of an easter egg. Spoiler warning for DELTARUNE Chapter 4.

## Customize
Use URL parameters to customize your message

**error:**
> An HTTPS Error code or a Chrome Error code. will override the message and be displayed with their official name.

**url:**
> Any URL, will be displayed at the bottom.

**message:**
> Any text you want to be displayed on the webpage. Will override any error message.

## Examples

The Default:

    https://xia-qi2450.github.io/dogcheck/

A 404 Error screen:

    https://xia-qi2450.github.io/dogcheck/?error=404

A 503 Error screen with a website attached:

    https://xia-qi2450.github.io/dogcheck/?error=503&url=https://github.com/Xia-Qi2450/dogcheck/

Dogcheck with custom message:

    https://xia-qi2450.github.io/dogcheck/?message=This%20is%20a%20custom%20message
